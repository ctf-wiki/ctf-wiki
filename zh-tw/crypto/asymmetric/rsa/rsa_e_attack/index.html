    公鑰指數相關攻擊¶

    小公鑰指數攻擊¶

    攻擊條件¶

    e 特別小，比如 e 爲 3。

    攻擊原理¶

    假設用戶使用的密鑰 e=3。考慮到加密關係滿足：

    c\equiv m^3 \bmod N

    則：

    \begin{align*} m^3 &= c+k\times N\\ m &= \sqrt[3]{c+k\times n} \end{align*}

    攻擊者可以從小到大枚舉 k，依次開三次根，直到開出整數爲止。

# coding=utf-8
import gmpy2
from Crypto.PublicKey import RSA
from multiprocessing import Pool
pool = Pool(4)

with open('./pubkey.pem', 'r') as f:
    key = RSA.importKey(f)
    N = key.n
    e = key.e
with open('flag.enc', 'r') as f:
    cipher = f.read().encode('hex')
    cipher = int(cipher, 16)


def calc(j):
    print j
    a, b = gmpy2.iroot(cipher + j * N, 3)
    if b == 1:
        m = a
        print '{:x}'.format(int(m)).decode('hex')
        pool.terminate()
        exit()


def SmallE():
    inputs = range(0, 130000000)
    pool.map(calc, inputs)
    pool.close()
    pool.join()


if __name__ == '__main__':
    print 'start'
    SmallE()

    題目¶

    RSA 衍生算法——Rabin 算法¶

    攻擊條件¶

    Rabin 算法的特徵在於 e=2。

    攻擊原理¶

    密文：

    c = m^2\bmod n

    解密：

    • 計算出 m_p 和 m_q：
    \begin{align*} m_p &= \sqrt{c} \bmod p\\ m_q &= \sqrt{c} \bmod q \end{align*}
    • 用擴展歐幾裏得計算出 y_p 和 y_q：
    y_p \cdot p + y_q \cdot q = 1
    • 解出四個明文：
    \begin{align*} a &= (y_p \cdot p \cdot m_q + y_q \cdot q \cdot m_p) \bmod n\\ b &= n - a\\ c &= (y_p \cdot p \cdot m_q - y_q \cdot q \cdot m_p) \bmod n\\ d &= n - c \end{align*}

    注意：如果 p \equiv q \equiv 3 \pmod 4，則

    \begin{align*} m_p &= c^{\frac{1}{4}(p + 1)} \bmod p\\ m_q &= c^{\frac{1}{4}(q + 1)} \bmod q \end{align*}

    而一般情況下，p \equiv q \equiv 3 \pmod 4 是滿足的，對於不滿足的情況下，請參考相應的算法解決。
